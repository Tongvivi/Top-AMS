## 多色打印原理

### 基本概念
多色3D打印的核心是**自动换料**。以打印红、蓝、绿三色模型为例：
1. 打印完红色部分后，系统自动退出红色耗材
2. 送入蓝色耗材，继续打印蓝色部分  
3. 打印完蓝色后，再次换成绿色耗材
4. 循环此过程直到打印完成

### TOP AMS换色机制
TOP AMS采用**热床温度信号**控制换色，工作流程如下：

1.**温度检测**：ESP32控制板实时监测打印机的热床温度变化

2.**换色触发**：假设我们有1、2、3、4号四种颜色的耗材
              当打印完1号颜色，需要切换到2号颜色时
切片软件会自动将热床温度设置为2°C作为换色信号
ESP32控制板检测到2°C信号后，进行退料, 然后控制2号电机进料2号耗材, 然后继续恢复打印

3.**循环换色**：当需要切换到3号、4号颜色时，同样设置对应的温度值（3°C、4°C），系统会自动识别并完成换色

**简单理解**：就是用热床温度当"暗号"——温度是几度就换成几号颜色，全程自动化无需人工干预。

## 系统结构

TOP AMS系统主要由两大模块组成：

### 1. 挤出机模块
- **数量**：支持1-8个挤出机（推荐新手从4个开始）
- **功能**：负责耗材的进给和退出
- **驱动**：每个挤出机配备独立的N20减速电机
- **控制**：一块驱动板可控制2个挤出机

### 2. 控制模块
- **主控**：ESP32开发板
- **驱动**：电机驱动板（数量根据挤出机数量确定）
- **功能**：
  - 实时监测热床温度
  - 控制进料/退料时序
  - 与打印机MQTT通信
  - Web界面管理

### 工作流程
```
热床温度变化 → ESP32检测 → 发送控制信号 → 驱动板工作 → 电机执行进/退料
```
